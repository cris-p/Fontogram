The above logo is based on two symbols: A and 🎞 (of particular Windows fonts). Fontogram is a method, WPF control type name, and library for generating a large variety of such icons for Microsoft .NET WPF applications, using a simple scheme for combining characters of any installed fonts with WPF Border shapes used as backdrop.

This screenshot of the included sample app shows more Fontogram examples:

You create an icon by creating an instance of the WPF control class 'Fontogram'. You specify a backdrop and a text, using any desired attribute values to modify them from their defaults. Multiple backdrop-text pairs can be specified in the same Fontogram, as different layers.

Why Fontogram

  • Useful: potentially good looking icons created with simple XAML/C#.
  • Removes complexity and repetition in creating a set of icons with WPF methods.
  • Minimal learning required: the creation of text+Border objects and the setting of control attributes are already familiar to any WPF developer.
  • Productivity: easy for WPF developers to think in terms of text and Border objects, with their specific attributes. The associated Fontogrammer app makes things even easier.
  • Quality: the operating system is very good at rendering fonts at any size.
  • Scalability: Fontograms are vector based, so no pixelation; antialiasing makes edges look smooth.
  • Color: works around WPF's inability to render emoji/symbol fonts in multicolor.
  • Flexibility: arbitrary number of backdrop and symbol objects; decent array of presentation and transformation parameters.

The provided sample code shows how to create custom Fontograms in XAML and reusable Fontogram packs in C# code. The code generated for the custom Fontograms includes comments that explain the control attributes.

What fonts

Windows 10 (which Fontogram should be used on) has some useful fonts - containing emojis, symbols, glyphs, object icons:

  • Segoe UI Emoji
  • Segoe MDL2 Assets
  • Webdings
  • Wingdings
  • Wingdings 2
  • Wingdings 3

The library and sample code are accompanied by the Fontogrammer tool, which displays a live preview of the Fontogram being designed, and generates the specific XAML and C# code that you need to use. Fontogrammer can also be used as documentation for Fontogram attributes - see the fields' tooltips.

When Fontogram

  • user interface needs some color or pictorial aids
  • need to display icons at arbitrary sizes - especially on the small side
  • an adequate icon library is not available
  • no help from a specialized icon designer available
  • must not spend much time on icons
  • simple monochrome font-based icons are not satisfactory
